# Source Packs

Place manually curated course files, personal notes and reference packs here.

Example:

```text
source-packs/
  cours-master-2025/
  pcg-anc-2026/
  ifrs-preparation-2027/
  iso-notes-personnelles/
```

Run:

```bash
pnpm ingest source-packs/cours-master-2025
```

The MVP creates a manifest only. Full Docling extraction and database indexing are planned for the next iteration.
