# Finance Learning Hub

## Goal
Private local-first learning hub for accounting, cost accounting, IFRS/IAS, ISO, finance and management control.

## Principles
- No external document connectors.
- All knowledge comes from local files or manually imported source packs.
- Every sourced answer must cite document, page, pack and date when available.
- Never mix course content and official reference content without saying so.
- Prefer deterministic retrieval before LLM reasoning.
- Generated exercises must include rubric, expected answer and competency tags.
- Corrections must separate calculation errors, accounting treatment errors and reasoning errors.
- ISO content must be handled as notes/checklists unless licensed text is explicitly allowed.

## Commands
- `pnpm dev`
- `pnpm lint`
- `pnpm test`
- `pnpm typecheck`
- `pnpm db:migrate`
- `pnpm ingest <path>`
